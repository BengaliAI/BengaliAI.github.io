Research

We are currently more interested in computer vision, speech and NLP based research. In parallel our teams work on Linguistic theory, language documentation, meme research and much more!

OOD-Speech: Bengali.AI Massively Crowdsourced Bengali Speech Recognition Project arXiv, 2023
project page arXiv

The first 2000 hour cross domain Bengali Speech Recognition Dataset from 25000 people.

BaDLAD: A Large Multi-Domain Bengali Document Layout Analysis Dataset ICDAR, 2023
project page arXiv

The first multidomain large Bengali Document Layout Analysis Dataset: BaDLAD. This dataset contains 33,695 human annotated document samples from six domains - i) books and magazines ii) public domain govt. documents iii) liberation war documents iv) new newspapers v) historical newspapers and vi) property deeds.

Bengali Grammatical Error Correction Project arXiv, 2023
project page Kaggle

Large Bengali grammatical error detection and correction project. Involves a novel linguist validated dataset with 100k+ samples. Hosted a Kaggle competition in 2023 to crowdsource solutions.
